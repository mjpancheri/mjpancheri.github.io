Resumo Profissional

Trabalho desde 2004 no desenvolvimento de sistemas Web. Trabalhei por 17 anos na coordenação e desenvolvimento de sistemas para educação a distância, com back-end e front-end em Java. Em 2019 comecei a estudar ReactJS e em 2021 coordenei um projeto de migração de front-end de Java para React. Atualmente estou atuando em projetos J2EE de API REST e SOAP. Nos últimos anos trabalhei em projetos de API REST em PHP/Laravel e microserviços em Java com Spring Boot, utilizando serviços em nuvem como RDS, ECS, S3, SQS, SNS, além de Docker, Datadog e NewRelic. Faço mentorias para ajudar os desenvolvedores menos experientes do time e grupos de estudos para aprimorar meus conhecimentos. Também tenho experiência em NodeJS, Ruby on Rails e Elixir/Phoenix. Realizei projetos de estudo sobre Kafka, RabbitMQ, Redis e GraphQL.
Gosto de desafios, de resolver problemas e sempre tive a preocupação de criar sistemas performáticos para proporcionar melhor experiência para os usuários.
